# `beetkeeper-plugin`

This is a peripheral [beets](https://beets.io/) plugin component for the [beetkeeper application](https://beetkeeper.dadbodaudio.com/). Pushes beets events automatically to the beetkeeper server for reactive workflows and import history tracking.

Only compatible in a `beetkeeper`-managed beets setup. `pip install beetkeeper` will install this distribution automatically.

## Official User Reference

Go to https://beetkeeper.dadbodaudio.com/ for all official `beetkeeper` user guides and docs.
