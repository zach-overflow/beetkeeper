# REST API Reference

The endpoint-by-endpoint reference below is generated from beetkeeper's OpenAPI schema. For a high-level
overview of what the API is for, see [Using the REST API](../usage/rest-api.md); for live, interactive docs
against your own instance, use Swagger UI at `/docs` on a running server.

!!! note "About the examples"
    Request and response examples are generated automatically from each operation's schema — they show the
    shape and types of the payloads. Treat the linked schema as the authoritative source for exact
    constraints and required fields.
